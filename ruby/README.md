# EcfDgii

Ruby SDK for integrating with the ECF SSD API (Dominican Republic Electronic Invoicing).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ecf-dgii'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install ecf-dgii
